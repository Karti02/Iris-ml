# 📖 Overview

This project is a Machine Learning web application built with **Python**, **Flask**, and **Scikit-Learn**.

The application predicts the species of an Iris flower based on four input measurements.

### Input Features

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

### Predicted Species

- 🌸 Iris Setosa
- 🌺 Iris Versicolor
- 🌼 Iris Virginica

The application features a clean and responsive interface, making it suitable for beginners learning Machine Learning deployment using Flask.
