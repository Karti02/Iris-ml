🌸 About Iris Flower Prediction

An intelligent Machine Learning application that predicts the species of an Iris flower based on its measurements.

📖 Project Overview

The Iris Flower Prediction System is a Machine Learning based web application that classifies an Iris flower into one of three species: Setosa, Versicolor, or Virginica. The prediction is made using four flower measurements:

  • 🌿 Sepal Length
  • 🌿 Sepal Width
  • 🌸 Petal Length
  • 🌸 Petal Width
Using these features, the trained ML model accurately predicts the flower species within seconds.

🌺 Dataset

The application uses the famous Iris Dataset containing 150 flower samples equally distributed among three species.

🤖 Machine Learning

The prediction model is trained using supervised learning algorithms from Scikit-Learn to achieve high prediction accuracy.

⚡ Fast Prediction

Users simply enter flower measurements and instantly receive the predicted Iris species.

⚙️ Technologies Used

HTML5 CSS3 Python Flask Scikit-Learn NumPy Pandas Machine Learning

🔄 How It Works

  1. User enters flower measurements.
  2. Input values are sent to the Flask backend.
  3. The trained ML model processes the data.
  4. The model predicts the Iris species.
  5. The prediction is displayed with confidence.

🎯 Project Goals

This project demonstrates how Machine Learning can solve real-world classification problems using simple numerical inputs. It also showcases the integration of Python, Flask, and Frontend Web Development into a complete intelligent web application.
